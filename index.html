PyTorch Jupyter Notebooks

A hands-on introduction to PyTorch and deep learning, starting from the fundamentals and progressing to modern neural network architectures.

View Repository

About

This repository contains a series of interactive Jupyter notebooks designed to teach PyTorch through practical examples. Each notebook builds on the previous ones, covering both theoretical concepts and implementation details.

Topics Covered

PyTorch Basics
Tensors, automatic differentiation, and basic operations.
Linear Regression
Building and training your first neural network.
Classification
Binary and multiclass classification models.
Data Loading
Datasets, DataLoader, batching, and preprocessing.
Convolutional Neural Networks
Image classification with CNNs.
Transfer Learning
Fine-tuning pretrained deep learning models.
Training & Evaluation
Loss functions, optimizers, metrics, and model evaluation.
Modern Deep Learning
Best practices for building scalable PyTorch projects.

Get Started

Browse the notebooks in order to progressively build your understanding of PyTorch and deep learning.
